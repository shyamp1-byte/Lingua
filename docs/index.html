Get started

Up and running in 5 minutes

1
Download & install
Open the DMG and drag Lingua into your Applications folder.

First launch only — macOS will block the app. Two ways to fix it:

Option A (easiest) — Terminal:
xattr -dr com.apple.quarantine /Applications/Lingua.app
Then open Lingua normally. Warning gone permanently.

Option B — System Settings:
Click Done on the warning → open System Settings → Privacy & Security → scroll down → click Open Anyway → enter your Mac password.
2
Get API keys (free tiers available)
Deepgram — transcription (200 hrs free/month)  ·  DeepL — translation (500k chars free/month)
3
Paste keys in ⚙ Settings
Open Lingua, click the gear icon, paste your keys. That's it — keys stay on your machine.
4
Grant Screen Recording permission
macOS will prompt you on first launch. Lingua uses this to capture system audio (not your screen content).
5
Hit ▶ Start Captions or press ⌘⇧L
Play any audio. Captions appear on the overlay within ~1 second.
